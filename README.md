# 🎯 InterviewGPT — AI-Powered Mock Interviewer

**InterviewGPT** is a sleek, premium, dark-themed AI mock interview simulator designed to help developers and CS candidates prepare for technical and behavioral interviews at top-tier companies.

👉 **Live Site:** [https://anaanfzd.github.io/ai-powered-mock-interviewer/](https://anaanfzd.github.io/ai-powered-mock-interviewer/) *(Optional: Enable GitHub Pages to view live)*

---

## ✨ Features

- **🎯 Role-Specific Customization:** Tailor interviews for Software Engineers, AI/ML Engineers, Data Analysts, Web Developers, DevOps, and more.
- **📈 Comprehensive Core CS Topics:** Interactive question banks covering:
  - Data Structures & Algorithms (DSA)
  - Object-Oriented Programming (OOP)
  - Database Management Systems (DBMS)
  - Operating Systems (OS)
  - Computer Networks (CN)
  - Web Development
  - Artificial Intelligence & Machine Learning
  - System Design
  - HR / Behavioral (STAR Method)
- **🧠 Adaptive Difficulty & Follow-up Questions:** Progression shifts dynamically (Easy ➡️ Medium ➡️ Hard) based on your answer accuracy, triggering follow-up questions from real interview formats.
- **📊 Detailed Answer Analysis & Scoring:** Get real-time grades out of 10 along with a breakdown of your strengths, areas of improvement, and the model ideal answer.
- **⏱️ Active Session Tracking:** Includes a visual scorecard tracking your average score, best score, topic-specific progress, accuracy/clarity/depth breakdown, and an active timer.
- **💻 Sleek UI:** Glassmorphism accents, fluid animations, responsive layouts, and interactive chat interface.

## 🛠️ Getting Started

To run the project locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/anaanfzd/ai-powered-mock-interviewer.git
   cd ai-powered-mock-interviewer
   ```
2. Start the local server:
   ```bash
   npm install
   npm start
   ```
3. Open **[http://localhost:3000](http://localhost:3000)** in your browser.
